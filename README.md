# Teslo API
1. Instalar las dependencias
```
yarn
```

2. Crear el arhivo __```.env```__ apartir del archivo __```.env.example```__

3. Levantar la DB
```
docker compose up --build
```

4. Levantar el modo desarrollo:
```
yarn start:dev
```

5. Ejecutar SEED
```
http://localhost:3002/api/seed
```
